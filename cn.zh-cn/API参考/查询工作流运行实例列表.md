>![](public_sys-resources/icon-note.gif) **说明：**   
>-   start大于等于1，最大1000，不设置则取默认值1。  
>-   limit取值最小1，最大1000，不设置则取默认值10。  
>-   name为只能包含字母、数字、“-”、“\_”的字符串。  
>-   graph\_urn为只能包含字母、数字、“-”、“\_”、“:”的字符串。  
>-   name和execution\_urn只能选择其一。
