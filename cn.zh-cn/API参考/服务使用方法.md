# 服务使用方法<a name="functiongraph_06_0210"></a>

公有云API符合RESTful API的设计理论。

REST从资源的角度来观察整个网络，分布在各处的资源由URI（Uniform Resource Identifier）确定，而客户端的应用通过URL（Unified Resource Locator）来获取资源。

URL的一般格式为：https://**_Endpoint_**/**_uri_**
