# 删除工作流触发器<a name="ZH-CN_TOPIC_0115410460"></a>

## 功能介绍<a name="zh-cn_topic_0020212651-chtext"></a>

删除一个工作流触发器。

>![](public_sys-resources/icon-note.gif) **说明：**   
>如果某个工作流引用了该触发器，则不能删除该触发器。  

## URI<a name="section7151914"></a>
