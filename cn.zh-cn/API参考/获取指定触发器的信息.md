**响应示例**

成功时的返回格式：

```
HTTP/1.1 200 OK 
{ 
    "trigger_id":"191e425a-924a-45d9-b730-38b5ffe339ad", 
    "trigger_type_code":"SMN", 
    "event_data": { 
        "topic_urn":"abc", 
        "subscription_status":"Unconfirmed" 
        }
}
```

失败时的返回格式（示例）：

```
HTTP/1.1 400 Bad Request 

{ 
  "error_code": "FSS.0400", 
  "error_msg": "Invalid trigger data" 
 }
```
