## 请求消息<a name="section59769852"></a>

**请求参数**

无

**请求示例**

```
GET /v1.0/7aad83af3e8d42e99ac194e8419e2c9b/fss/functions/urn:fss:cn-north-1:7aad83af3e8d42e99ac194e8419e2c9b:function:default:test:latest/code HTTP/1.1
```
